# Fashion & Style - Proyecto E-Commerce

Proyecto web desarrollado por Kelvin Piñero, Jonathan Heredia, Yulieth Galarza y Carlos Lema como parte de la carrera de Ingeniería en Sistemas (PUCE, 2025).

## Descripción
Fashion & Style es una tienda online de moda juvenil, moderna y responsiva.  
Permite a los usuarios explorar productos, solicitar crédito directo y gestionar un carrito de compras con una interfaz limpia y funcional.

El sitio incluye:
- Página principal con hero responsivo y panel de búsqueda.
- Tienda con productos destacados.
- Sección “Sobre nosotros” con misión y visión.
- Formulario de crédito directo validado con jQuery.
- Footer con enlaces y redes sociales.
- Diseño totalmente adaptable a diferentes dispositivos.

## Tecnologías utilizadas
- HTML5  
- CSS3 / Bootstrap 5.3  
- JavaScript (jQuery)  

## Características principales
- Diseño responsivo para computadoras, tablets y móviles.  
- Validación de formularios con jQuery.  
- Navegación fluida con anclas y efectos suaves.  
- Estructura modular del código.  
- Paleta de colores en tonos beige y verde, con estilo elegante y minimalista.
